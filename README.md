# ফলহাট — তাজা ফলের বাজার (PWA)

অফিসারদের ওয়াইফদের হোয়াটসঅ্যাপ গ্রুপের জন্য তৈরি একটি ছোট প্রোগ্রেসিভ ওয়েব অ্যাপ। বিক্রেতা (এডমিন) ফলের পোস্ট দেন, ক্রেতারা পরিমাণ অর্ডার করে, মন্তব্য ও রেটিং দেয়।

## ফিচার

- **ক্রেতা ভিউ:** সব পোস্ট দেখা, পরিমাণ অর্ডার করা, মন্তব্য + ১-৫ স্টার রেটিং দেওয়া, লাইভ "অবশিষ্ট পরিমাণ" দেখা।
- **বিক্রেতা (এডমিন) ভিউ:** নতুন পোস্ট তৈরি/সম্পাদনা/মুছে ফেলা, পোস্ট বন্ধ-চালু করা, প্রতিটি পোস্টের সব অর্ডার ও মন্তব্য দেখা, মন্তব্যের উত্তর দেওয়া।
- উপরে ডান কোণার আমের ছবিতে চাপলে পাসওয়ার্ড পপআপ আসে — সঠিক পাসওয়ার্ড দিলে বিক্রেতা ভিউ খোলে। সাধারণ দৃষ্টিতে এটি স্রেফ একটি ছবি বলেই মনে হবে।
- **বিক্রেতার পাসওয়ার্ড:** `USTisha99` — এটি বদলাতে চাইলে `js/app.js` ফাইলের শুরুতে `ADMIN_PASSWORD` লাইনটি বদলান।
- Firebase Firestore দিয়ে রিয়েল-টাইম শেয়ারিং — বিক্রেতা পোস্ট দিলে সবার ফোনে সাথে সাথে দেখা যাবে।
- PWA: মোবাইলে "Add to Home Screen" করে অ্যাপ আইকনের মতো ব্যবহার করা যাবে, অ্যাপ-শেল অফলাইনেও খুলবে (তবে নতুন পোস্ট/অর্ডার দেখতে ইন্টারনেট লাগবে)।

## ধাপ ১ — নিজের Firebase প্রজেক্ট তৈরি করুন (একবার করতে হবে)

1. https://console.firebase.google.com -এ যান, **"Add project"** চেপে একটি প্রজেক্ট বানান (যেমন নাম দিতে পারেন `folhaat`)। Google Analytics লাগবে না, বন্ধ রেখে দিতে পারেন।
2. বাম পাশের মেনু থেকে **Build → Firestore Database** এ যান, **"Create database"** চাপুন, **Production mode** বেছে নিন, নিকটতম লোকেশন (যেমন `asia-south1` বা `asia-southeast1`) বেছে নিন।
3. **Project settings** (⚙️ আইকন) → নিচে স্ক্রল করে **"Your apps"** সেকশনে **Web (</>)** আইকনে চাপুন → অ্যাপের একটা নাম দিন (যেমন `folhaat-web`) → **Register app**।
4. যে `firebaseConfig = {...}` অবজেক্টটি দেখাবে, পুরোটা কপি করুন।
5. এই প্রজেক্টের `js/firebase-config.js` ফাইল খুলে, ভেতরের `firebaseConfig` অবজেক্টের জায়গায় কপি করা অবজেক্টটি বসিয়ে দিন।

## ধাপ ২ — Firestore সিকিউরিটি রুলস বসান

Firestore Console-এ **Rules** ট্যাবে গিয়ে নিচের রুলস বসিয়ে **Publish** করুন:

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /posts/{postId} {
      allow read: if true;
      allow create, update, delete: if true;

      match /comments/{commentId} {
        allow read, create, update: if true;
      }
      match /orders/{orderId} {
        allow read, create: if true;
      }
    }
  }
}
```

> **সিকিউরিটি নোট:** এই রুলস সবার জন্য খোলা, কারণ এই অ্যাপে আলাদা লগইন সিস্টেম (Firebase Authentication) নেই — বিক্রেতা প্যানেল শুধু একটি পাসওয়ার্ড পপআপ দিয়ে আড়াল করা, যা টেকনিক্যালি দেখলে পুরোপুরি সুরক্ষিত নয় (কেউ কোড দেখলে পাসওয়ার্ড পেয়ে যেতে পারে)। ছোট, বিশ্বস্ত একটি গ্রুপের জন্য এটি যথেষ্ট, কিন্তু বড় পরিসরে বা স্পর্শকাতর তথ্যের জন্য ব্যবহার করলে Firebase Authentication যুক্ত করা উচিত — দরকার হলে এটাও পরে করিয়ে দিতে পারব।

## ধাপ ৩ — লোকালি টেস্ট করুন

`index.html` ফাইলটা সরাসরি ব্রাউজারে খুললে কিছু ব্রাউজারে Firestore বা সার্ভিস ওয়ার্কার কাজ না-ও করতে পারে। তার বদলে একটা সাধারণ লোকাল সার্ভার চালান:

```
cd folhaat
python3 -m http.server 8080
```

তারপর ব্রাউজারে `http://localhost:8080` খুলুন।

## ধাপ ৪ — GitHub Pages-এ ডিপ্লয় করুন

1. GitHub-এ একটা নতুন রিপোজিটরি বানান (যেমন `dev-atikuzzaman/folhaat`)।
2. এই ফোল্ডারের সব ফাইল সেই রিপোতে পুশ করুন।
3. রিপোর **Settings → Pages** এ গিয়ে Branch হিসেবে `main` এবং ফোল্ডার `/ (root)` বেছে **Save** করুন।
4. কিছুক্ষণ পর `https://dev-atikuzzaman.github.io/folhaat/` লিংকে অ্যাপটি চালু হয়ে যাবে — এই লিংকটাই হোয়াটসঅ্যাপ গ্রুপে শেয়ার করবেন।

## ফাইল গঠন

```
folhaat/
├── index.html          মূল পেজ (ক্রেতা + বিক্রেতা ভিউ, সব মোডাল)
├── manifest.json        PWA মেনিফেস্ট
├── sw.js                সার্ভিস ওয়ার্কার (অফলাইন অ্যাপ-শেল ক্যাশ)
├── css/style.css        সম্পূর্ণ ডিজাইন (রঙ, ফন্ট, কার্ড, মোডাল)
├── js/firebase-config.js   আপনার Firebase কনফিগ এখানে বসাবেন
├── js/app.js             সব লজিক — অর্ডার, মন্তব্য, রেটিং, এডমিন গেট
└── icons/                জেনারেট করা আম-আইকন (PWA আইকন + হেডারের গোপন বাটন)
```

## কাস্টমাইজ করতে চাইলে

- **ফলের তালিকা বদলানো:** `js/app.js`-এর শুরুতে `FRUIT_OPTIONS` ও `FRUIT_EMOJI` অবজেক্ট এডিট করুন।
- **একক (কেজি/ডজন ইত্যাদি) বদলানো:** `UNIT_OPTIONS` অ্যারে এডিট করুন।
- **রঙ/থিম বদলানো:** `css/style.css`-এর শুরুতে `:root` এর ভেরিয়েবলগুলো বদলান।
- **পাসওয়ার্ড বদলানো:** `js/app.js`-এ `ADMIN_PASSWORD` বদলান।

কোনো অংশ পরিবর্তন বা নতুন ফিচার (যেমন WhatsApp-এ সরাসরি অর্ডার মেসেজ পাঠানো, একাধিক বিক্রেতা, ছবি আপলোড) লাগলে বলবেন — যুক্ত করে দিতে পারব।
